# 💖 Valentine Website

An interactive Valentine-themed React web application featuring animated hearts, confetti effects, dynamic button behavior, and smooth UI transitions.

Built using React with functional components and CSS animations.

## 🚀 Features

- 💕 Floating animated hearts background
- 🎉 Confetti animation on YES click
- 📈 Dynamic scaling YES button
- 🏃 Moving NO button interaction
- 🎨 Gradient background UI
- ✨ Custom CSS animations
- 📱 Responsive design

## ⚙ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/aksmisr/valentine-website.git
```

### 2️⃣ Navigate into the project folder

```bash
cd VALENTINE-SITE
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Run the development server

```bash
npm start
```

The app will open in your browser at:

```
http://localhost:3000
```

---

## 🧠 How It Works

The app uses:

- React `useState` for screen navigation
- Conditional rendering for multi-step UI
- CSS keyframe animations for floating hearts and confetti
- Dynamic inline styles for interactive buttons

---

## 🎨 Animations Used

- floatUp
- fallDown
- fadeIn
- pulse
- bounce
